Au démarrage de MinUSc un modèle de quartz est chargé dans la fenêtre de visualisation.
Vous pouvez :
  • Déplacer le modèle dans l'espace en utilisant la souris
  • Modifier le mode d'affichage
  • Sélectionner une partie du modèle sur laquelle appliquer les modifications
  • Charger un autre modèle
